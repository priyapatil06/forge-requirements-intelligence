# Contributing

1. Fork the repository and create a feature branch.
2. Keep domain packs in `backend/app/domain_packs.py`.
3. Add schema changes to both Pydantic models and frontend TypeScript types.
4. Add or update tests for prompt/output changes.
5. Run backend tests, frontend tests, and the production frontend build before opening a pull request.

Do not commit API keys, Jira tokens, customer requirements, or production data.
